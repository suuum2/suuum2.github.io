Skills and Tools

  • HTML5

    시맨틱 태그를 이용하여 레이아웃 코드를 작성하며, 웹 표준과 웹 접근성을 준수하고 검색엔진 최적화를 고려한 코드 작성을 위해 노력합니다.

  • Editor

    최근까지 EditPlus와 Visual Studio Code를 사용하여 코드를 작성하였으며 NotePad++ 도 사용합니다. Git에 포트폴리오를 등록해 보았습니다.

  • PHOTOSHOP & ILLUSTRATOR

    포토샵과 일러스트를 활용하여 시안을 제작하고 디자인 작업을 합니다.
